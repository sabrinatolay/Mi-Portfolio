Sobre mí

Soy Sabrina Tolay, desarrolladora Full Stack recién egresada de la Tecnicatura Universitaria en Desarrollo Web. Durante mi carrera trabajé con tecnologías como PHP, Python, Java, CodeIgniter y MySQL, desarrollando aplicaciones web completas desde el diseño de la base de datos hasta la interfaz de usuario.

Soy muy lógica, lo que me permite ir desde el detalle más pequeño para entender por qué falla lo más grande. No me quedo tranquila hasta que todo funciona correctamente — si hay un problema, sigo pensando hasta encontrarlo. Cada bug resuelto no es solo una corrección: es experiencia acumulada y la satisfacción de haber contribuido con algo real.

Me interesa especialmente la experiencia del usuario: me gusta detectar qué falla en una interfaz y mejorarlo para que tanto el trabajador que usa una app internamente como el usuario final tengan exactamente lo que necesitan. Busco una empresa donde pueda aportar esa mirada, crecer dentro de un equipo y ser parte de cambios que se noten.

Proyectos

Estos son algunos de los proyectos que desarrollé durante mi carrera. Cada uno representó un desafío distinto y una oportunidad concreta de aprender.

DG contrucciones y soluciones

DG es un sistema para un Maestro Mayor de Obras, que permite administrar solicitudes de trabajo, publicar trabajos realizados , gestionar equipos para obras y recibir calificaciones de clientes. Incluye una sección pública para que usuarios visualicen trabajos y envíen solicitudes. Este proyecto resuelve la necesidad de centralizar la gestión de trabajos y la comunicación entre el profesional, clientes y trabajadores.

  • PHP
  • MySQL
  • Bootstrap
Ver código en GitHub

Tareasapp_PHP-Collaborator

PHP Collaborator es una plataforma para gestión de tareas y colaboradores en proyectos PHP usando modelo MVC en CodeIgniter 4. Donde se permite asignar subtareas automáticamente a colaboradores disponibles,confiando en su competencia.

  • CodeIgniter
  • MySQL
  • JavaScript
Ver código en GitHub

Mi-Veterinaria_Happy-Feets

Happy Feets es una aplicación web para la gestión integral de una veterinaria, desarrollada con PHP utilizando el patrón MVC de CodeIgniter 4. Permite registrar, modificar y eliminar información sobre amos, mascotas y veterinarios, así como llevar el control de visitas veterinarias. Pensada para facilitar el trabajo administrativo del personal.

  • PHP
  • MySQL
  • HTML / CSS
